W3M3 - Word Count using MapReduce

학습 목표

Demonstrate your understanding of Apache Hadoop by writing and running a MapReduce job. This project will help you gain hands-on experience with Hadoop's MapReduce framework by building a word counter program and running it on a large text file.
기능요구사항

MapReduce Program:

The program should correctly read input text from HDFS.
The Mapper should emit key-value pairs for each word in the text.
The Reducer should aggregate these key-value pairs and output the total word counts.
Job Execution:

The MapReduce job should run without errors on the Hadoop cluster.
The job should process the entire e-book and produce an accurate word count.
The job should utilize Hadoop's distributed processing capabilities to handle the large input file efficiently.
Result Output:

The output should be a text file stored in HDFS.
Each line of the output file should contain a word and its corresponding count, separated by a tab or space.
The output should be retrievable and readable from HDFS.
프로그래밍 요구사항

MapReduce Program:

Write a MapReduce program to count the occurrences of each word in a text file.
Implement the Mapper and Reducer classes in Python.
The Mapper should process each line of the text and emit key-value pairs where the key is a word and the value is 1.
The Reducer should aggregate the values for each key (word) and output the total count for each word.
Setup Hadoop Environment:

Ensure that your Hadoop environment is correctly set up, either on a single-node or multi-node cluster.
Upload your chosen e-book (over 200 pages in English) to the Hadoop Distributed File System (HDFS).
Running the MapReduce Job:

Package your MapReduce program into a JAR file or a script, depending on the language used.
Submit the MapReduce job to the Hadoop cluster.
Monitor the job's progress through the Hadoop web interface or command-line tools.
Result Collection:

Retrieve the output of the MapReduce job from HDFS.
Ensure the output contains the word counts for the e-book.
예상결과 및 동작예시

MapReduce Job:

A functional MapReduce program that successfully processes a large e-book and counts the occurrences of each word.
The job should run to completion without errors and efficiently utilize the Hadoop cluster resources.
Word Count Output:

An output file in HDFS containing word counts for the entire e-book.
Each word in the e-book should have an accurate count in the output file.
Documentation:

Comprehensive documentation that guides users through compiling, running, and verifying the MapReduce job.
Clear instructions on handling HDFS operations related to the input and output files.
Include steps for uploading the input file to HDFS and retrieving the output file.
Document how to interpret the output and verify the results.
Submission:

Submit the source code for the MapReduce program.
Provide a README file with step-by-step instructions for compiling the program, uploading the input file to HDFS, running the MapReduce job, and retrieving the output file.
Include the name and source of the chosen e-book.
