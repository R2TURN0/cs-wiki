- 한 주간 배운 내용을 주말에 복습해야겠습니다.
  - 수업 자료를 다시 읽어보겠습니다.
  - SQL 쿼리 작성을 좀 더 심층적으로 실습해보겠습니다.
  - ERD 작성 툴을 이용하여 실습을 해보겠습니다.
  - 복습 이후, 제출한 과제들을 다시 살펴보겠습니다.
- 기록을 좀 더 체계적이고 구체적으로 해야겠습니다.
- 맥북 사용법을 익혀야겠습니다.
